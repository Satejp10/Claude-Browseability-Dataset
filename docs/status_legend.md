# Status Legend

## Access status examples

| Status | Meaning |
|---|---|
| `Confirmed Claude-specific block` | Robots or policy explicitly references Claude/Anthropic agents |
| `Confirmed broad bot block` | Site broadly blocks unapproved crawlers or all non-whitelisted bots |
| `Likely blocked / unreliable` | Direct access often fails or returns incomplete content |
| `Login-gated` | Requires account/session for meaningful content |
| `Paywalled / subscriber-gated` | Full content requires payment or subscription |
| `Dynamic / JS-heavy` | Server-side fetch may not see rendered content |
| `Anti-bot protected` | CAPTCHA, Cloudflare, WAF, rate-limiting, or bot checks interfere |
| `Mixed` | Some public pages work; others fail or degrade |

## Live fetch impacted values

| Value | Meaning |
|---|---|
| `Yes` | Claude-style user-directed retrieval is likely affected |
| `Partial` | Some pages may work, but important content often fails or is incomplete |
| `Unclear` | Evidence is not enough to judge live fetch impact |
| `No` | No known live-fetch issue, but included for tracking/context |

## Workaround patterns

| Barrier | Best workaround |
|---|---|
| Reddit/forum comments | Paste thread/comments or export text |
| X/LinkedIn/social post | Paste text or upload screenshot |
| YouTube/video | Provide transcript or captions |
| Paywall | Paste article text you have legitimate access to |
| Login-gated docs | Export/download/upload the file |
| JS-heavy page | Screenshot, PDF print, or copy visible text |
| Retail page | Use current manual price/availability or a shopping tool |
